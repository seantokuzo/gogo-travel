# @gogo/mobile

GoGo Travel mobile app — Expo SDK 57 + expo-router, part of the pnpm workspace.

## Quickstart

```bash
pnpm install                       # from the repo root
pnpm --filter @gogo/mobile start   # Expo dev server (press i / a / w)
pnpm --filter @gogo/mobile ios     # or jump straight into the iOS simulator
```

Routes live in `src/app/` (file-based routing via expo-router).
See the root `README.md` for the project overview.
